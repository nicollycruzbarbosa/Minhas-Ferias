Atividades para realizar nas férias

Além das atividades típicas, você também pode aproveitar essas:

  • Brinque de fazer roupas:Uma sugestão é brincar de fazer tie-dye (camisetas hippies cheias de cores). Para isso, basta comprar tintas para tecido e seguir as instruções.

  • Aprenda origami: Para dias chuvosos, que tal aprender a fazer origamis? Existem muitos tutoriais na internet que podem ajudar.

  • Monte peças teatrais:Que tal montar uma releitura de uma peça que vocês assistiram juntos?Ou quem sabe criar uma história nova?

  • Faça uma noite do cinema (com direito a pipoca):Todo mudo já fez uma “noite do cinema”. Aproveite as férias para caprichar nessa atividade. Se ainda não tiver, compre embalagens de pipoca iguais aquelas das salas de cinema e crie o ambiente dentro de casa.

  • Faça um picnic:As férias são uma excelente oportunidade de juntar os amigos da escola para uma atividade conjunta diferente.
